# endsWith

Develop a function endsWith(string, search[, length]) that returns true if the
string ends with search, else false.

If present, length represents the length of the string.

# Test Case(s)

`Argument 1` - "Hello?"
`Argument 2` - "?"
`Return Value` - true

`Argument 1` - "Hello?"
`Argument 2` - "o"
`Return Value` - false

`Argument 1` - "Hello?"
`Argument 2` - "o"
`Argument 3` - 5
`Return Value` - true
