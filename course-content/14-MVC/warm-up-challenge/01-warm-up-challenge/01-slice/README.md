# slice

Develop a function slice(start[, end]) creates a new array containing element range array[start]
through array[end-1] and returns it.

The Big O for the implementation should be O(N).

# Test Case(s)

`Input 1` - [1,2,3]
`Input 2` - 1
`Input 3` - 2
`Output return` - [2]

`Input 1` - [1,2,3]
`Input 2` - 0
`Input 3` - 2
`Output return` - [1,2]

`Input 1` - [1,2,3]
`Input 2` - 2
`Output return` - [3]

`Input 1` - [1,2,3]
`Input 2` - -2

`Input 2` - -3

`Input 1` - [1,2,3]
