# includes

Develop a function includes(array, element) that returns true if element is an element of the array
else false.

The Big O for the implementation should be O(N).

# Test Case(s)

`Input 1` - ["cat","dog","frog","snake"]
`Input 2` - "snake"
`Output return` - true

`Input 1` - ["cat","dog","frog","snake"]
`Input 2` - "cat"
`Output return` - true

`Input 1` - ["cat","dog","frog","snake"]
`Input 2` - "pancake"
`Output return` - false
